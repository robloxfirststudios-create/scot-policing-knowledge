# Police Scotland Knowledge Base

Welcome to the Police Scotland Knowledge Base — a draft site to collect guidance, reference material, and signposts for policing topics relevant to Scotland.

**Status:** Draft — not legal advice. Always consult primary legislation, Crown Office & Procurator Fiscal Service (COPFS) guidance, and your force legal advisers for authoritative advice.

## Navigation

- Laws and Regulations
  - Criminal Damage (Scotland)

## How to use this site

- This site is a starting point for officers, staff, and trainees. Pages are drafted and intended to provide high-level context and links to primary sources.
- If you spot errors or want to contribute, please open a pull request in this repository.

## Local preview

Requirements: Python 3.8+ and pip

Install the Material theme for MkDocs:

pip install mkdocs-material

Serve locally:

mkdocs serve

Build for production:

mkdocs build
